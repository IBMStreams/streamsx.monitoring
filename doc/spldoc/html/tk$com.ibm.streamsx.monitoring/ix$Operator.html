Operators

JobStatusMonitor
This operator uses JobStatusSource and prevents near duplicate notifications being emitted.
JobStatusSource
The JobStatusSource operator uses the JMX API to retrieve status updates from one or more jobs, and provides status changes as tuple stream.
LogSource
The LogSource operator uses the JMX API to retrieve log events and provides log messages as tuple stream.
MetricsMonitor
The MetricsMonitor operator connects to the MetricsSource operator and receives incoming metric tuples.
MetricsSource
The MetricsSource operator uses the JMX API to retrieve metrics from one or more jobs, and provides metric changes as tuple stream.
SystemMonitorSource
The Java Operator SystemMonitorSource collects the CPU and Memory usage from the current running system and emits a tuple in an interval.
